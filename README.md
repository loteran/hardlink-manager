# Hardlink Manager

A web application for managing hardlinks in Docker containers.

## Features

- Create and manage hardlinks
- Web-based interface
- Docker support

## Setup

1. Clone the repository
2. Build the Docker image: docker build -t hardlink-manager .
3. Run with Docker Compose: docker compose up -d

## Configuration

See docker-compose.yml for configuration options.
